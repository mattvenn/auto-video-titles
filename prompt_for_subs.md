Analyse the subtitle file and suggest title card placements for animated lower-third overlays.

There are two card types:

Standard card — 2 lines, place when a new subject begins or a key term/tool is first introduced.

Call to action card — 3 lines, place sparingly at natural end-of-video moments only.

Guidelines:

- Don't over-saturate , only where it adds value
- line1 and line2 max 20 chars, line3 max 30 chars — hard VFD display constraint
- duration_s default 4.0 unless context suggests the moment needs more time
- Confident placements: enabled = true. Uncertain: enabled = false so the reviewer can decide without deleting the entry
- Use notes to briefly explain the placement and flag anything ambiguous
